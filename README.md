# Logcock - app for viewing logs and other files

Logcock is web app thaht you can use with following features:
 * file browsing
 * file downloading
 * for tailing a log file
 * define multiple browsing file/directory system roots


Tehnology stack:
 - Project Reactor
 - Spring Webflux
 - Spring Boot
 - Thymeleaf
 - Bulma
 - Fontawsome
 - RequireJS
 - Turbolinks
 - Stimulus
